Seatwork COMP 009 · OOP · 2025

Seatwork 1 — Operators

This activity involves the manual tracing and calculation of various Java operator expressions to understand how the language handles arithmetic, logic, and bitwise data at the machine level.

Java Operators Manual Tracing Bitwise Logic Arithmetic Precedence Increment & Decrement

Overview

What this seatwork covers

The activity covers the precise execution order and logic of multiple operator categories, including arithmetic, relational, logical, and assignment operators. It specifically tests manual proficiency in tricky increment/decrement cases and bitwise operations, requiring students to predict the exact output without the aid of a compiler.

Reflection

What I learned

Manually calculating these operator expressions was a real test of my attention to detail, especially when dealing with the "tricky" increment and decrement logic that modifies variables mid-statement. Since I’m transitioning from C, I’m familiar with the math, but Java’s strictness with boolean results and its specific hybrid model for handling bytecode means there’s zero room for error in these manual traces. It was a lowkey challenge to keep track of memory changes on paper without just hitting "run" in an IDE, but it really deepened my understanding of how the JVM processes logic behind the scenes. Meeting the exact requirements for Part II—like hitting precisely six outputs with a mix of bitwise and logical operators—felt like solving a puzzle that reinforces the "disciplined" architecture Java is known for.
